# starbucks-clone
보러가기 : https://jangddoll.github.io/starbucks-clone/<br>
가용 기기: PC, 태블릿, 휴대폰

# 프로젝트에 사용한 기술
* HTML / CSS
* JavaScript
* swiper
* GSAP
* GSAP ScrollTrigger
